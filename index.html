QUALIFICATIONS

What We’re Looking For

HTML, CSS, JavaScript Basics

A foundational understanding of core web development languages is essential.

Familiarity with Frameworks

Experience with tools like React or Tailwind CSS is a plus, but not required.

Problem-Solving Skills

The ability to debug, optimize, and think critically about code challenges.

Willingness to Learn

An eagerness to adapt to new technologies and grow your skill set.

Communication Skills

Clear, concise communication for collaborating with teams and mentors.

Teamwork

Collaborative spirit to work well in group projects and contribute effectively.
